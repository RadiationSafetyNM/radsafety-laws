■ 원자력안전법 시행규칙 \[별표 1\] \<개정 2019. 12. 3.\>

|                                                                               |          |
|-------------------------------------------------------------------------------|----------|
| <u>방사성동위원소등 사용자에 대한 정기검사의 시기</u>(제77조 관련)            |          |
| 구분                                                                          | 검사시기 |
| 1\. 방사성동위원소등을 인체에 사용하는 공장 또는 사업소                       |          |
| 가. 인체의 치료 및 체내외 검진을 목적으로 사용하는 경우                       | 매 1년   |
| 나. 가목 외의 경우                                                            |          |
| 1\) 연간 사용량이 3.7기가베크렐(100밀리퀴리) 이상                             | 매 3년   |
| 2\) 연간 사용량이 3.7기가베크렐(100밀리퀴리) 미만                             | 매 5년   |
| 2\. 방사성동위원소등을 인체에 사용하지 않는 공장 또는 사업소                  |          |
| 가. 밀봉된 방사성동위원소                                                     |          |
| 1\) 연간 사용량이 111테라베크렐(3,000퀴리) 이상                               | 매 3년   |
| 2\) 연간 사용량이 111테라베크렐(3,000퀴리) 미만                               | 매 5년   |
| 나. 밀봉되지 않은 방사성동위원소                                              |          |
| 1\) 연간 사용량이 3.7기가베크렐(100밀리퀴리) 이상                             | 매 3년   |
| 2\) 연간 사용량이 3.7기가베크렐(100밀리퀴리) 미만                             | 매 5년   |
| 다. 방사선발생장치                                                            |          |
| 1\) 1기가전자볼트 이상                                                        | 매 1년   |
| 2\) 1기가전자볼트 미만, 1메가전자볼트 이상                                    | 매 3년   |
| 3\) 최대 전압 350킬로볼트 이상으로서 용량이 350킬로볼트, 5밀리암페어 1대 이상 | 매 3년   |
| 4\) 최대 전압 350킬로볼트 미만으로서 용량이 250킬로볼트, 5밀리암페어 2대 이상 | 매 3년   |
| 5\) 3) 또는 4)에서 정한 각각의 전압ㆍ용량 및 수량에 해당되지 않는 경우        | 매 5년   |
| 3\. 방사성동위원소등의 이동사용을 전문으로 하는 사업소                        | 매 1년   |
| 4\. 판매허가를 받은 자                                                        |          |
| 가. 밀봉된 방사성동위원소                                                     |          |
| 1\) 연간 판매량이 370테라베크렐(10,000퀴리) 이상                              | 매 1년   |
| 2\) 연간 판매량이 370테라베크렐(10,000퀴리) 미만                              | 매 3년   |
| 나. 밀봉되지 아니한 방사성동위원소                                            |          |
| 1\) 연간 판매량이 37테라베크렐(1,000퀴리) 이상                                | 매 1년   |
| 2\) 연간 판매량이 37테라베크렐(1,000퀴리) 미만                                | 매 3년   |
| 다. 방사선발생장치                                                            | 매 3년   |
| 5\. 방사성동위원소등의 생산허가를 받은 자                                     | 매 1년   |
| 6\. 업무대행자                                                                | 매 1년   |
